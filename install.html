Install VisitorLogic

Paste the following snippets into your website.

1. Configuration (optional)

Place this block before the runtime script. Edit only the values. 

<script>
window.VisitorLogic_CONFIG = {
  industry: "",  // change this to your actual website industry e.g agriculture, fintech, health etc
  audience: "",        // change this to your actual target audience e.g teams, founders, enterprises etc
  page_type: ""     // change this to your actual page type e.g homepage, pricing, features, landing etc
};
</script>

If you omit the above required parameters, VisitorLogic will silently fail.

2. Runtime

Add this script immediately after the configuration. 

<script
  src="https://cdn.jsdelivr.net/gh/investor-uyah/visitor-logic/dist/visitorlogic.v1.js"
  defer>
</script>

3. Mark elements to personalize

VisitorLogic updates only elements with these attributes. 

<h1 data-vl-headline>Default headline</h1>
<p data-vl-subheadline>Default subheadline</p>
<button data-vl-cta>Get started</button>

Your existing text remains visible if no segment matches.
